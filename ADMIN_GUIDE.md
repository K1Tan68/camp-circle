# 管理画面ガイド

サイト公開中：**https://camp-circle.vercel.app**

## 🔐 ログイン方法

1. `https://camp-circle.vercel.app/sign-in` にアクセス
2. メールアドレスで登録またはログイン
3. 管理画面へ → `/admin` ページが自動で開く

**※ 初回：招待リンクまたは管理者に申請が必要です**

---

## 📸 ギャラリー機能（複数写真管理）

### 写真を追加
1. 管理画面 → **「ギャラリー」** タブをクリック
2. **「📸 写真をアップロード」** セクションで以下を入力：
   - **写真を選択** ← クリックして画像ファイルをアップロード
   - **日付** ← 撮影日
   - **場所** ← 撮影場所（例：道志の森、北海道など）
   - **タイトル・キャプション** ← 写真の説明（例：夕暮れのテントサイト）
   - **説明（任意）** ← 詳しい説明（例：夏合宿での一コマ）
3. **「アップロード」** ボタンをクリック

### 写真を削除
- ギャラリー一覧で、削除したい写真にマウスを重ねる
- 右上の **ゴミ箱アイコン** をクリック
- 確認で削除完了

### スマホでの操作
- **タップして画像を選択** → 自動プレビュー表示
- **スマホサイズに最適化** → ボタンが大きく、タッチしやすい配置
- **横スクロール対応** → 小さい画面でも全機能使える

---

## 📝 活動報告機能（ブログ投稿）

### 記事を追加
1. 管理画面 → **「ブログ」** タブをクリック
2. 右上の **「+ 追加」** ボタンをクリック
3. 投稿フォームに入力：
   - **タイトル** ← 記事のタイトル（必須）
   - **タグ** ← 「活動報告」「ノウハウ」「お知らせ」「イベント」から選択
   - **本文** ← 記事の本文（改行は \n で表現、または複数行ペースト）
   - **抜粋** ← 記事一覧に表示される短い説明（省略すると本文の最初 80 字）
4. **「公開する」** チェックボックス
   - ☑️ = サイトに公開される
   - ☐ = 下書き保存（サイトに表示されない）
5. **「保存」** をクリック

### 記事を編集
- ブログ一覧で、編集したい記事の右側 **✏️ ボタン** をクリック
- フォームが開く → 変更 → **「保存」**

### 記事を削除
- 記事の右側 **🗑️ ボタン** をクリック
- 確認で削除完了

### スマホでの操作
- **フォーム全体がスマホ画面に収まる**
- **本文入力は大きなテキストエリア** → タイプしやすい
- **ボタンも大きく** → ワンタップで確定

---

## 🎯 PC・スマホ両対応

### PC（デスクトップ）
- 左サイドバーで各機能を選択
- グリッド表示で複数写真一覧を表示
- テーブル形式で記事一覧を表示

### スマホ（タブレット・携帯）
- サイドバーが自動折りたたみ → **タブバーに変更**
- ボタン・入力欄が **タッチしやすいサイズ**
- グリッドが **1 列または 2 列に自動調整**
- 画像プレビュー **スワイプ対応**

## 💡 使用例

### ケース 1：キャンプ直後に写真をアップ
```
日付：2026-06-26
場所：富士山麓キャンプ場
タイトル：初夏の焚き火タイム
説明：全員で肝試しをした後の焚き火。お互いに童心に帰った雰囲気で、キャンプの良さを改めて実感。
→ スマホからもすぐにアップロード可能！
```

### ケース 2：月例活動の報告をブログで発信
```
タイトル：6月の月例活動「道志の森キャンプ」を実施しました
本文：
6月15日〜16日、桃山学院大学キャンプサークルは...
（複数段落で詳しく記載）
タグ：活動報告
公開：☑️ チェック済み
→ サイトのブログセクションに自動反映！
```

### ケース 3：緊急のお知らせを配信
```
タイトル：【重要】7月の活動スケジュール変更のお知らせ
本文：悪天候予報のため、予定を変更いたします...
タグ：お知らせ
公開：☑️ チェック済み
→ 下書き保存して、部員に確認してから公開もOK
```

## 🔐 権限について

- **Editor（編集者）** ← ギャラリー・ブログの追加・編集・削除のみ可
- **Admin（管理者）** ← 全機能 + メンバー管理 + 招待管理

## ❓ よくある質問

**Q. 写真のサイズに制限ありますか？**
A. ありません。大きなサイズでも自動圧縮されます。

**Q. 画像は何枚までアップロードできますか？**
A. 制限なし。複数写真でギャラリーを充実させてください。

**Q. 下書き保存した記事はどこに？**
A. ブログ一覧で「○ 下書き」表示。ログアウト後も保存されます。

**Q. スマホで編集中にネット が切れたら？**
A. 自動保存はないので、**こまめに「保存」をクリック**してください。

**Q. 管理画面の URL は？**
A. `https://camp-circle.vercel.app/admin`

---

## 📞 トラブル時

- **ログインできない** → メールアドレスが正しいか確認。初回は招待リンク必要。
- **画像がアップロードできない** → ファイルサイズ確認（通常数 MB まで OK）
- **保存がうまくいかない** → ブラウザの再読込（Ctrl+R または Cmd+R）
