# Design Direction — キャンプサークル Website

## Concept
自然の中にぽつんとテントが佇むフラットイラスト風。温かみのあるアースカラー。インスピレーション元：Wild Journey風のアウトドアアプリデザイン。

## Color Palette
- `--color-forest`: #2D4A2D (深緑 — primary)
- `--color-moss`: #4A6741 (草緑 — secondary)
- `--color-earth`: #8B7355 (土 — accent warm)
- `--color-sand`: #D4B896 (砂 — light accent)
- `--color-sky`: #C4D4B0 (空緑 — background light)
- `--color-dusk`: #3D5A3E (夕暮れ緑)
- `--color-orange`: #D4793A (オレンジ — CTA accent)
- `--color-cream`: #F5F0E8 (クリーム — background)
- `--color-dark`: #1C2B1C (テキスト)

## Typography
- **Heading**: "Playfair Display" — serif, 自然の重厚感
- **Body**: "Lato" — sans-serif, 読みやすさ
- **Accent**: "Merriweather" — italic for pull quotes

## Layout & Style
- フラットイラスト風の背景（SVGシェイプで山・木・テント）
- 大きなヒーローセクション（暗い森の背景にテント）
- セクションは緩やかな曲線区切り（wave dividers）
- カードはシャドウ少なめ、ボーダーやアースカラーで区切る
- ネガティブスペース多め

## Sections
1. **Hero** — 大きな見出し + テント背景 + CTAボタン
2. **About** — サークル紹介
3. **Schedule** — 活動スケジュール
4. **Gallery** — 写真グリッド
5. **Blog** — 活動報告カード
6. **Join** — 入部案内 + お問い合わせフォーム
7. **Footer** — SNSリンク
